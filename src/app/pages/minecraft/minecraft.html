Install the Modpack

  1. Download the CurseForge app (or use Prism Launcher)
  2. Open the Kenro's Anarchy modpack page and click Install
  3. Wait for the modpack to download and install
  4. Launch the modpack from your launcher
  5. Continue with How to Connect below

How to Connect

  1. Open Minecraft Java Edition
  2. Click Multiplayer from the main menu
  3. Click Add Server
  4. Enter a name (e.g. Kenro's Server)
  5. Enter the server address: kenro.tech
  6. Click Done, then select the server and click Join Server
